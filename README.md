CPerl-Chat
========

C + Perl Chat

### Dependency

libwww-perl

perl

libgtop2-dev

libncursesw5-dev

libncurses5-dev

expect

### Debian Ubuntu 
```
sudo apt-get install libwww-perl perl libgtop2-dev libncursesw5-dev libncurses5-dev expect
```

### Install & Execute 

```
cd CPerl-Chat
make && sudo make install
./client/cperl-chat
```
