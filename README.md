CPerl-Chat
========

C + Perl Chat

### Dependency

libwww-perl

perl

libgtop2-dev

libncursesw5-dev

libncurses5-dev


### Debian Ubuntu 
sudo apt-get install libwww-perl perl libgtop2-dev libncursesw5-dev libncurses5-dev

### Install & Execute 

git clone https://github.com/eldin77/CPerl-Chat

cd CPerl-Chat

make && sudo make install

./client/cperl-chat
